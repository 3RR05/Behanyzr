About Behanyzr

Behanyzr helps to analyze Behaviour shifts by scraping data from reddit and NewsAPI along with visual graph presentation, The one who sees this can modify this project if any logical problems wanders in it.

💡 How It Works

1
Collect

Scrape Reddit posts or News articles via PRAW(Python Reddit API Wrapper) and NewsAPI.

2
Process

Process the collected data by cleaning texts and remove symbols.

3
Analyze

Apply VADER sentiment analysis and TF-IDF(Like converting the data into Dataframe to perform query actions) topic modeling across time windows.

4
Visualize

Generate Matplotlib/Seaborn charts and statistical drift reports.
